# WildLand

## 所需环境

vue-cli 3.x及以上

node 10.x及以上

## 运行方法

```
将仓库克隆下来
npm install
npm install --save element-plus
npm install @element-plus/icons-vue
npm install --save vue-router
npm run serve
```

## 文件结构

| 文件夹     | 放置内容                 |
| ---------- | ------------------------ |
| page       | 登录页、管理员页、网站页 |
| views      | 具体的页面内容           |
| components | 组件                     |
